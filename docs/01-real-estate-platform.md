## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Maps**: Google Maps API or Mapbox
- **Database**: Supabase or Firebase
- **Images**: Next.js Image + Cloudinary
- **Search**: Client-side filtering or Algolia
- **Forms**: React Hook Form
- **Deployment**: Vercel

## Core Pages
1. Home (featured properties)
2. Property listings (grid/map view)
3. Property detail page
4. Advanced search
5. About the agency
6. Contact page
7. Agent profile pages
8. Saved properties (favorites)

- Address and location
- Price

- Square footage
- Property type (house, condo, etc.)
- Features/amenities
- Year built
- HOA fees
- Description
- Photo gallery
- Virtual tour link

- Property type
- Bedrooms/bathrooms
- Square footage range
